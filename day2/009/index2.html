thuộc tính class trong html
        thuộc tính class được sử dụng để gom các thẻ html vào 1 nhóm theo ý muốn
        thẻ html có thể chung class
        khi mà có nhiều class trong 1 thẻ thì viết cách nhau ra ví dụ
        class="class1 class2 class3"
