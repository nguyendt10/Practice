Các cách viết tắt của margin
        Trường hợp 1 nếu 4 margin khác nhau
        margin: margin-top value margin-right value margin-bottom value margin-left value
        ví dụ margin: 10px 20px 30px 40px
        Trường hợp 2: nếu 3 margin bằng nhau
        margin: margin-chung
        ví dụ: margin:20px (cả margin 4 hướng là 20px)
        Trường hợp 3: margin-top bằng margin bottom và margin-left bằng margin-right
        margin: margin-top margin left;
        margin: 10px 20px;(margin-top=margin-bottom; margin-left = margin-right)
        Trường hợp 4: margin-top # margin-bottom nhưng margin-left = margin-right
        margin: top left bottom;
        margin:10px 30px 60px;
        Chú ý: nếu 1 trong 4 hướng là 0 thì không cần viết đơn vị
        Ví dụ :margin:0;
margin:15px 7px 8px 9px; => margin-top:15px; margin-right:7px; margin-bottom: 8px; margin-left: 9px
margin: 12px 6px 15px; => margin-top:12px; margin-right = margin-left: 6px; margin-bottom:15px;
margin: 20px 40px; => margin-top = margin-bottom :20px; margin-left = margin-right: 40px;
margin: 50px; => margin-top = margin-right = margin-bottom = margin-left: 50px;
