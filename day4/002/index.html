Có thể gộp
        padding-top, padding-left, padding-right, padding-bottom
        vào 1 thuộc tính duy nhất là padding
        padding:20px; giống margin
    
        quy tắc viết tắt của padding giống hệt margin
TH1: padding:30px => padding-top:30px; padding-right:30px; padding-bottom:30px; padding-left:30px;
TH2: padding:30px 10px 6px 20px; => padding-top:30px; padding-right:10px; padding-bottom:6px; padding-left:20px;
TH3: padding:5px 9px 12px; => padding-top:5px; padding-right:9px; padding-bottom:12px; padding-left:9px;
TH4: padding:10px 20px; => padding-top:10px; padding-right:20px; padding-bottom:10px; padding-left:20px;
